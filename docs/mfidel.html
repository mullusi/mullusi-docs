Mfidel

Atomic Fidel Boundary

Mfidel treats each fidel as a complete symbolic unit. Shape, sound, and identity are not split into root letters or internal components for processing.

Atomicity Rule

  • Each fidel is atomic.
  • Atomic shape maps to atomic fidel identity.
  • Atomic fidel identity maps to atomic whisper sound.
  • Vowel overlays are sound-layer fusions, not structural decomposition.

Grid Contract

f[row][col]
row in 1..34
col in 1..8

The grid reference is an addressing system. It must not be used to decompose a fidel into lower structural units.

Prohibited Models

Pattern Decision
Root-letter extraction Blocked
Consonant-plus-vowel token splitting Blocked
Codepoint decomposition or recomposition Blocked
Sound overlay as internal structure Blocked
